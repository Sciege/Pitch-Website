#  CCS ProjectHub  
A web-based platform for students of the **College of Computer Studies (CCS)** at **MSU-IIT** to submit, showcase, and explore programming projects across different major subjects.

---

##  Overview  
**CCS ProjectHub** is designed to make project submissions more interactive and transparent.  
Students can upload their outputs for each lab activity, view the works of their classmates, and gain inspiration from others.  
Instructors can also view submissions, grade them, and highlight top projects — all in one place.

---

##  Features  
-  **Student Dashboard:** Submit and manage project outputs  
-  **Instructor View:** View and grade student submissions  
-  **Project Showcase:** Browse and view peer projects  
-  **Search & Filter:** Easily find projects by subject or activity  
-  **Live Links:** Visit deployed projects directly (e.g., via GitHub Pages or Vercel)

---

##  Tech Stack  
- **Frontend:** HTML, CSS (pure, no frameworks)  

---

##  Goal  
To create a collaborative environment where CCS students can **learn, share, and grow** by showcasing their coding projects in one platform.
